## Run Locally

Clone the project

```bash
  git clone https://github.com/DevShaunB/fem-article-preview-component.git
```

Go to the project directory

```bash
  cd fem-article-preview-component
```

Run `index.html`

```bash
  <browsername> index.html
```

E.g.

```bash
  firefox index.html
```

```bash
  google-chrome index.html
```
